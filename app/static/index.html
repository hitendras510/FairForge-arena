Fairness Metrics

Rigorous, Standard
Bias Measurement

Disparate Impact (DI)
≥ 0.80 = Fair

Ratio of favorable outcome rates between unprivileged and privileged groups. The 80% rule from EEOC guidelines.

0.0 (Severe)0.8 (Fair)1.0 (Perfect)
Statistical Parity Difference
≈ 0.0 = Fair

Absolute difference in selection rates. Measures overall representation equity across groups.

Equal Opportunity Difference
≈ 0.0 = Fair

Difference in true positive rates. Ensures equally qualified individuals get equal positive outcomes.

Core Innovation

Reinforcement Learning
That Trains Fairness In

Unlike post-hoc bias mitigation, FairForge uses PPO to retrain the model's decision policy itself — rewarding accuracy while penalizing unfairness.

1

Custom OpenAI Gym Env

We wrap your classification model in a Gymnasium-compatible environment where each "step" is a batch of predictions.

env = FairnessEnv(model, data)
obs = env.reset()
action = policy(obs)
obs, reward, done, info = env.step(action)
2

Dual-Objective Reward

The reward function balances prediction accuracy with a fairness penalty — creating a Pareto-optimal trade-off.

R = α · accuracy
  - β · |DI - 1.0|
  - γ · SPD
3

PPO Policy Optimization

Stable-Baselines3 PPO agent learns to adjust prediction thresholds per group, converging to a fair policy.

model = PPO("MlpPolicy", env)
model.learn(total_timesteps=50k)

Bias Explanations

FairForge XAI — powered by Gemini Online
Why is the model biased against Hispanic female applicants?

Analysis shows a Disparate Impact of 0.41 for Hispanic females — significantly below the 0.80 threshold.

Root causes:

1. Training data underrepresents this group (only 6.2% of samples)

2. Feature "zip_code" acts as a proxy for ethnicity

3. Historical hiring patterns encoded in labels

Recommendation: Run PPO training with α=0.7, β=0.3 to rebalance without significant accuracy loss.
